Two distinct sub-scopes share this leaf module:
- `src/styles.css` is the single entry point for the application's stylesheet. It imports Tailwind v4 via `@import "tailwindcss" source(none)` with explicit `@source` directives pointing at `../src/routes`, `../src/components/shopify`, and `../src/lib` so Tailwind can scan class names across those directories. A `@theme inline` block maps CSS custom properties (`--color-*`, `--font-*`) to design tokens defined in `:root`, and a `@custom-variant dark (&:is(.dark *))` enables a dark-mode variant. Global resets live in `@layer base`, while the interactive hero-range panel animation lives in `@layer components`. A `prefers-reduced-motion` media query disables animations for accessibility.
- `public/` holds immutable static files served directly by the web server: `site.webmanifest` declares the PWA (standalone display, brand colors), `robots.txt` disallows authenticated/account/search routes, and `sitemap.xml` enumerates the site's top-level pages with priorities and last-modified dates for SEO.
